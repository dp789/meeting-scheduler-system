# meeting-scheduler-system

## Architecture

```
    .
    └── controllers
    └── db
```

## Pre-requisites

- yarn

## Steps to setup DB locally

- ./setup-local.sh

create a new database and run migrations for you locally

## Steps to run application locally

- yarn dev
